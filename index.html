IEM Market Tracker V1

How to use this tracker

Use the tracker as a secondary-market reference for IEM pricing. Search for an IEM, check its estimated Market Value, then click into the individual page to inspect the underlying listings and decide whether the estimate looks trustworthy.

Market Value

Uses a 90-day average when there are at least 3 clean samples. If recent data is sparse, it uses a recency-weighted fallback so newer listings count more than older listings.

Liquidity / Confidence

The same score appears on the main page and each individual IEM page. It combines included sample count, recency, recent sample depth, source diversity, and price consistency into a 0–100 badge.

High Medium Low

Deal indicators

Deal labels compare a listing price against the current market value. Treat them as leads to inspect, not automatic buying advice.

  • This tool is still in development, so matching errors, missed bundles, stale statuses, and pricing mistakes can happen.
  • Always click through to the source listing and verify condition, accessories, seller history, currency, and whether the item is actually still available.
